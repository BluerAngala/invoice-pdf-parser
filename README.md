# 发票识别统计工具

一个纯前端的发票识别、统计和管理工具,支持批量处理、智能去重、金额统计、排版打印等功能。

## 功能特性

- 📁 **批量上传**: 支持一次性上传多张发票图片
- 🔍 **OCR识别**: 自动识别发票号码、金额、日期等关键信息
- 🔄 **智能去重**: 基于发票号码自动检测并标记重复发票
- 💰 **金额统计**: 自动计算所有有效发票的金额合计
- 🖨️ **排版打印**: 生成规范的发票统计表格,支持直接打印
- 📥 **导出功能**: 支持导出为PDF和Excel格式
- 🎨 **可视化界面**: 直观的卡片式布局,方便查看和管理

## 技术栈

- Vue 3 + TypeScript
- Vite
- Tesseract.js (OCR识别)
- jsPDF (PDF导出)
- html2canvas (页面截图)

## 安装使用

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 构建生产版本

```bash
npm run build
```

## 使用说明

1. **添加发票**: 点击"添加发票文件"按钮,选择一张或多张发票图片
2. **开始识别**: 点击"开始识别"按钮,系统会自动识别所有发票信息
3. **查看结果**: 识别完成后,可以在列表中查看每张发票的详细信息
4. **去重功能**: 开启"发票去重"开关,系统会自动标记重复的发票
5. **导出数据**: 
   - 点击"打印发票"直接打印统计表格
   - 点击"下载PDF"导出为PDF文件
   - 点击"导出Excel"导出为CSV格式

## 注意事项

- 首次使用时,Tesseract.js会下载中文语言包,可能需要一些时间
- 建议上传清晰的发票图片以获得更好的识别效果
- OCR识别可能存在误差,建议识别后检查关键信息
- 所有数据处理都在浏览器本地完成,不会上传到服务器
