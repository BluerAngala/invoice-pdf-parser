# 发票识别统计工具

一个纯前端的 PDF 发票识别、统计和管理工具，支持批量处理、智能去重、金额统计、导出清单等功能。

🔗 **在线体验**: [https://invoice-pdf-parser.netlify.app](https://invoice-pdf-parser.netlify.app)

## 功能特性

- � **PD去F 发票识别**: 自动解析 PDF 发票，提取发票号码、金额、日期等关键信息
- � **批量统上传**: 支持选择文件、选择文件夹、拖拽上传多种方式
- � **多页版 PDF 支持**: 自动拆分多页 PDF，识别每页发票
- � **智能去重**:  基于发票号码自动检测并标记重复发票
- 💰 **金额统计**: 实时计算有效发票的金额合计
- 📊 **导出清单**: 导出 Excel 清单 + 去重后的 PDF 文件打包下载
- 🔒 **隐私安全**: 纯前端处理，数据不上传服务器

## 技术栈

- Vue 3 + TypeScript
- Vite
- PDF.js（PDF 解析）
- ExcelJS（Excel 导出）
- JSZip（文件打包）

## 安装使用

### 1. 安装依赖



### 3. 构建生产版本

```bash

```

## 使用说明

1. **上传发票**: 点击「选择文件」或「选择文件夹」按钮，或直接拖拽 PDF 文件到上传区域
2. **自动识别**: 系统自动解析 PDF 并识别发票信息，显示处理进度
3. **查看结果**: 左侧列表显示所有发票，点击可预览和编辑详情
4. **智能去重**: 自动标记重复发票，点击「智能去重」可一键删除
5. **导出清单**: 点击「导出清单」下载 ZIP 包，包含：
   - Excel 发票清单（按日期排序，带序号）
   - 去重后的 PDF 文件（文件名带序号，方便对照）

## 注意事项

- 目前仅支持文本型 PDF 发票，图片型 PDF 暂不支持
- 识别结果仅供参考，建议核对关键信息
- 所有数据处理都在浏览器本地完成，保护数据隐私
